**1. stat-card** — Big number + label + optional context
- pattern.html: A 400x300px container (just big enough to show the component). Contains:
  - `.stat-number` — giant number, font-size 120px, font-weight 900, color #ffffff
  - `.stat-label` — uppercase label, font-size 14px, font-weight 500, color rgba(255,255,255,0.5), letter-spacing 0.1em
  - `.stat-context` — fine print, font-size 16px, color rgba(255,255,255,0.4) (optional)
  - Placeholder text: "94%", "ACCEPTANCE RATE", "Across all PSP integrations"
- README.md fields:
  ```json
  [
    { "type": "text", "sel": ".stat-number",  "label": "Stat Value",   "mode": "text", "rows": 1 },
    { "type": "text", "sel": ".stat-label",   "label": "Stat Label",   "mode": "text", "rows": 1 },
    { "type": "text", "sel": ".stat-context",  "label": "Context",     "mode": "text", "rows": 1 }
  ]
  ```
  - Composes into: stat-hero-single, data-dashboard, quarterly-report
  - When to use: Post leads with a single compelling metric
  - Not for: Posts that lead with emotion before data
