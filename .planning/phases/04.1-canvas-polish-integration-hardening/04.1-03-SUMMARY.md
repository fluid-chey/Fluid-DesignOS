## Decisions Made
- stdin MUST be 'inherit' not 'pipe' for claude spawn -- piped stdin causes indefinite hang (GitHub issue #771)
- Concurrent generation uses simple lock (reject with 409) rather than queueing -- single user canvas doesn't need queue
- SSE frames use distinct event types: default data for stdout, 'stderr' for stderr forwarding, 'done' for completion
- Session ID sent as first SSE frame so client can track which session is being generated
